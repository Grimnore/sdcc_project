# SDCC Learning Project

A personal workspace for getting comfortable with SDCC (Small Device C Compiler) and 8051 microcontroller programming.

## Setup
- Compiler: SDCC
- Target: 8051
- Editor: VS Code

## Compile & Generate Hex
```cmd
sdcc -mmcs51 main.c
packihx main.ihx > main.hex
```
